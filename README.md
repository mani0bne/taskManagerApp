# Task Manager App
A  Task Management application built with React.
This app allows users to create, update, delete, and move tasks across different stages with a clean and responsive UI.

## Features
Add new tasks with title, description, priority status and due date.

## Tech Stack

Frontend: React (Vite)

Styling: Tailwind CSS

State Management: React Hooks

Storage: Browser LocalStorage
